# CJ training bot

This is an interactive learning bot designed for people who want to study new topics, reinforce their knowledge, and test themselves through quizzes.

## How to use (commands)

### `/start`

This command is used to start a conversation with the bot.

**Example**

`/start`

### `/help`

This command will give you an instructions about how to use the bot.

**Example**

`/help`

### `/learn url`

Using this command, you can provide the bot with a URL to an article, documentation page, blog post, or any other learning source you want to study and remember. The bot will analyze the content, generate a short and easy-to-understand summary, and then create an interactive quiz to help you test your understanding and reinforce the material.

**Example**

`/learn https://www.rockstargames.com/games/sanandreas`

### `/quiz`

After the bot has learned one or more topics, you can use this command to challenge yourself by taking a quiz on a specific subject.

**Example**

`/quiz`
